## Start local

### Install dependencies

- [rust](https://www.rust-lang.org/tools/install)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)
- [nodejs](https://nodejs.org/en/)
- yarn `npm install -g yarn`
- Run `yarn`

### Run

```sh
yarn dev
```
